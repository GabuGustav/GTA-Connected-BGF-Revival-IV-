Configure Game Installation

Choose ONE of the following options:

Option 1: Install ZMenu (Recommended)

Install ZMenu directly in the root of your GTA IV folder:

Download ZMenu
  1. Open the GTA Connected launcher
  2. Go to Tools → Game Settings
  3. Select Grand Theft Auto IV from the game dropdown
  4. Click Browse next to Game EXE Path
  5. Select GTAIV.exe inside the root folder

Option 2: Use Retail Version

Download and extract retail version to your root folder:

Retail 1080 (Recommended) Retail 1070
  1. Extract the retail version to your GTA IV root folder
  2. Open the GTA Connected launcher
  3. Go to Tools → Game Settings
  4. Select Grand Theft Auto IV from the game dropdown
  5. Click Browse next to Game EXE Path
  6. Select GTAIV.exe inside the new Retail folder

Default path: C:\Program Files (x86)\Steam\steamapps\common\Grand Theft Auto IV\GTAIV\Retail\GTAIV.exe

Final Setup

  1. Go to Tools → Launcher Settings
  2. Set your player name
  3. Double-click a server in the server browser to connect
