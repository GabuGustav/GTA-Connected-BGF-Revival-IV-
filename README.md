# BGF Revival IV

A modern, responsive website for the BGF Revival IV gaming server.

## Features

- **Modern Design**: Glass morphism effects with gradient backgrounds
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Real-time Updates**: Simulated server status and player statistics
- **Interactive Elements**: Smooth scrolling, hover effects, and animations
- **Server Information**: Easy connection details with copy-to-clipboard functionality
- **Player Leaderboard**: Top players display with rankings
- **Social Links**: Footer with social media integration

## Technologies Used

- **HTML5**: Semantic markup for accessibility
- **Tailwind CSS**: Utility-first CSS framework for styling
- **JavaScript**: Interactive features and real-time updates
- **Font Awesome**: Icons for UI elements
- **Google Fonts**: Custom typography (Orbitron for gaming feel)

## Getting Started

1. Open `index.html` in your web browser
2. The website will load immediately with all features functional
3. No server setup required - it's a static website

## Customization

### Server Information
Update the server details in the `index.html` file in the Connect section:
- Server IP: Change `play.gtconnected.com`
- Port: Change `25565`
- Version: Update as needed

### Styling
- Modify colors by editing the gradient classes in the HTML
- Adjust glass effect intensity in the CSS
- Change fonts by updating the Google Fonts import

### Real-time Updates
- Update intervals in `script.js` (currently 5 seconds for status, 10 seconds for player data)
- Modify the simulation logic to connect to actual server APIs
